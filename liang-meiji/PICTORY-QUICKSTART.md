# Pictory AI 快速上手 - 5分钟出片

## 一、准备工作（5分钟）

### 1. 注册账号
- 访问：https://pictory.ai
- 用Google账号登录（最快）
- 选择套餐：**Standard ($29/月)** 先试用

### 2. 选择模式
登录后，选择：**Script to Video**  
（这是我们要用的模式：文案+音频 → 视频）

---

## 二、制作第一条视频（15分钟）

### Step 1: 上传文案和音频

**方案A：直接上传音频（推荐）**
1. 点击 "Upload Audio"
2. 选择 `01-sakamoto-ryoma.mp3`（英文音频）
3. Pictory会自动转写字幕
4. 点击 "Proceed"

**方案B：粘贴文案**
1. 复制 `01-sakamoto-ryoma-tts.txt` 内容
2. 粘贴到文本框
3. 点击 "Proceed"

---

### Step 2: AI自动生成（3-5分钟）

Pictory会自动：
- 分析文案，识别关键词（Sakamoto, Ryoma, samurai, Japan, Meiji）
- 从素材库匹配相关图片/视频
- 生成字幕并同步时间轴
- 添加转场效果

**你不需要做任何事，等待即可。**

---

### Step 3: 人工审核优化（5-10分钟）

#### 3.1 预览视频
点击播放按钮，看整体效果。

#### 3.2 替换不合适的画面
如果某个场景的图片不对：
1. 点击该场景（timeline下方的缩略图）
2. 点击右侧 "Browse scenes"
3. 搜索关键词（例如 "samurai sword" / "Japanese castle"）
4. 选择更合适的图片/视频
5. 点击 "Replace"

#### 3.3 调整字幕样式（可选）
1. 点击 "Captions" 标签
2. 选择字幕位置（Bottom / Center / Top）
3. 选择字体样式（推荐：Arial Bold，白色字+黑色描边）
4. 调整字体大小

#### 3.4 添加背景音乐（可选）
1. 点击 "Audio" 标签
2. 点击 "Add Music"
3. 选择分类：Epic / Cinematic / Documentary
4. 试听后点击 "Add"
5. 调整音量（Background volume: 20-30%，不要盖过旁白）

---

### Step 4: 导出视频（3-5分钟）

1. 点击右上角 "Generate"
2. 选择分辨率：**1080p**（必须，YouTube标准）
3. 选择格式：MP4
4. 点击 "Generate Video"
5. 等待3-5分钟，Pictory会发邮件通知
6. 下载视频到本地

## 四、常见问题

### Q1: 素材库的图片质量怎么样？
**A:** Pictory内置Shutterstock/Unsplash免版权库，质量很高。如果觉得不满意，可以手动搜索替换。

### Q2: 可以上传自己的图片吗？
**A:** 可以。点击 "Uploads" → 上传你收集的历史图片（浮世绘、老照片）→ 拖到时间轴替换。

### Q3: 字幕精度如何？
**A:** 英文字幕精度很高（95%+），基本不需要修改。如果有错字，点击字幕直接编辑。

### Q4: 可以调整画面停留时间吗？
**A:** 可以。点击场景 → 拖动时间轴边缘调整长度。

### Q5: 导出后发现有问题怎么办？
**A:** 重新进入项目（Projects → 选择项目）→ 修改 → 重新导出（不扣额度）。

收集高质量历史图片，上传到Pictory：

后续视频直接套用，省50%审核时间。

同时开5个标签页，并行处理5条视频：
- Tab 1: 上传+AI生成
