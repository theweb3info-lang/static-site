## 1. 产品概述

### 1.1 产品名称建议

| 方案 | 名称 | 英文 | 说明 |
|------|------|------|------|
| A | **字迹** | ZiJi | 双关：字的笔迹 + 学字的轨迹 |
| B | **笔小友** | BiPal | 儿童友好，拟人化 |
| C | **写字星球** | HanziWorld | 适合低龄用户获取感 |

> 推荐方案 A「**字迹**」——简洁、有质感，成人与儿童皆可接受；英文名 **ZiJi** 全球可注册。

### 1.2 Slogan

> **一笔一画，写出你的字迹。**

### 1.3 核心价值

- **说一个字，看它怎么写**：语音输入 → 笔画动画，零门槛。
- **跟着写，越写越好**：AI 跟写打分，实时纠正笔顺与结构。
- **简体·繁体·日语，一个 App 搞定**：三套权威笔顺数据，无缝切换。
- **大人小孩各有所爱**：儿童模式趣味激励，成人模式高效学习。

## 2. 目标用户与场景

### 2.1 用户画像

| 用户群 | 年龄 | 典型场景 | 核心诉求 |
|--------|------|----------|----------|
| 🧒 学龄前/小学低年级 | 4-9 岁 | 课后练字、亲子互动 | 趣味引导、正确笔顺、成就激励 |
| 🎓 海外华人子女 | 5-14 岁 | 周末中文学校补充 | 母语维持、标准笔顺、拼音辅助 |
| 🌏 中文/日语学习者 | 16+ 岁 | 自学备考 HSK/JLPT | 系统分级、遗忘曲线、听写检验 |
| 👨‍🏫 教师/培训机构 | — | 课堂教学、布置作业 | 班级管理、进度追踪、批量授权 |

### 2.2 使用场景

- **碎片时间**：通勤/排队时完成「每日 5 字」
- **课堂辅助**：教师投屏演示笔顺动画
- **亲子时光**：家长与孩子 PK 写字
- **考试备考**：HSK/JLPT 分级字库系统复习

## 3. 字形支持

| 字形 | 数据源 | 字数覆盖 | 特殊处理 |
|------|--------|----------|----------|
| **简体中文** | Make Me a Hanzi + 国标笔顺 | ~9,500 字 | GB2312 全覆盖 |
| **繁体中文** | 台湾教育部笔顺数据库 | ~4,808 字（常用） | 支持注音符号标注 |
| **日语汉字** | KanjiVG | ~6,500 字 | 标注音读/训读、JLPT 等级 |

> **三语切换**：同一汉字（如「学/學」）可一键对比不同字形的笔顺差异。

## 4. 用户模式

### 4.1 儿童模式 🧒

- 大按钮、圆角、手绘插画 UI
- 动画吉祥物引导（IP 待设计）
- 简化功能入口，突出「今日任务」
- 写完一字 → 星星/烟花/撒花动画奖励
- 家长锁（设置/付费需家长验证）

### 4.2 成人模式 🎓

- 紧凑信息密度 UI，水墨极简风
- 支持整句输入 → 逐字拆解笔顺
- 读音释义、组词造句、字源演变
- 快捷手势操作，效率优先

### 4.3 模式切换

- Onboarding 时选择，后续可在设置中随时切换
- 切换即时生效，无需重启

## 5. 功能需求

### 5.1 P0 — 核心功能（MVP 必须）

#### 5.1.1 语音输入

| 项目 | 说明 |
|------|------|
| 输入方式 | 点击麦克风按钮 → 说出汉字/词语/句子 |
| 识别引擎 | 系统原生 STT（iOS Speech / Android SpeechRecognizer） |
| 语言支持 | 普通话、粤语（P1）、日语 |
| 降级方案 | 识别失败 → 弹出键盘手动输入 |
| 补充输入 | 键盘输入（拼音/注音/罗马字）、手写识别输入（P1） |

#### 5.1.2 笔画动画

- SVG 矢量笔画，逐笔描绘动画
- 可调速：0.5x / 1x / 2x
- 暂停/单步/重播控制
- 每笔高亮时同步显示笔画名称（横、竖、撇、捺、点…）
- 笔画顺序编号标注

#### 5.1.3 三语切换

- 顶部 Tab 或下拉切换：简体 / 繁体 / 日语
- 切换后笔顺数据、读音、释义同步更新
- 同字不同笔顺时显示差异提示

#### 5.1.4 模式切换

- 设置页开关：儿童 ↔ 成人
- 切换影响：UI 主题、功能入口、激励形式、内容展示深度

#### 5.1.5 跟写打分

| 项目 | 说明 |
|------|------|
| 书写区域 | 米字格（默认）/ 田字格（可切换） |
| 引导 | 半透明笔顺提示 → 用户跟写 |
| 评分维度 | 笔顺正确性（40%）、笔画形态（30%）、结构比例（30%） |
| 容错机制 | 允许 ±15% 偏差；连续错误 2 笔后温柔提示（非红叉），如"试试从这里开始 ✨" |
| 提示风格 | 儿童：语音 + 动画角色鼓励；成人：文字简洁提示 |
| 回放对比 | 写完后可回放自己的书写过程 vs. 标准笔顺，叠加对比 |
| 评分展示 | 星级（1-5 星）+ 百分制 + 各维度雷达图（成人模式） |

### 5.2 P1 — 重要功能

#### 5.2.1 学习体系

- **分级字库**
  - 简体：人教版 1-6 年级 / HSK 1-6 级
  - 繁体：台湾国小 1-6 年级
  - 日语：JLPT N5-N1
- **每日任务**：根据当前级别推送 5-20 个新字 + 复习字
- **遗忘曲线**：基于艾宾浩斯算法安排复习间隔（1天/3天/7天/14天/30天）
- **听写模式**：App 读音 → 用户书写 → 自动批改 → 显示正确笔顺
- **进度报告**：每周/月报告，含掌握字数、正确率趋势、薄弱笔画分析

#### 5.2.2 部首与语言教学

- **部首教学**：拆解偏旁 → 展示同部首字 → 理解造字逻辑
- **组词造句**：学完单字展示常用词组和例句
- **标准发音**：拼音/注音/假名读音，点击即播放
- **笔画名称**：每一笔标注中文名称（横、竖、撇、捺、提、折…）

#### 5.2.3 智能功能

- **易错字提醒**：统计用户高频错误笔画，针对性强化
- **同音字对比**：输入拼音列出同音字，对比字形差异
- **笔画数搜索**：输入笔画数 → 列出对应汉字
- **手写识别输入**：用手指/触控笔书写 → 识别为汉字 → 展示标准笔顺

### 5.3 P2 — 增值功能

#### 5.3.1 内容增值

- **字源动画**：甲骨文 → 金文 → 小篆 → 楷书演变动画
- **字体切换**：楷书（默认）、行书、圆体预览
- **字帖 PDF 生成**：选择字库 → 生成带田字格的可打印 PDF 练习帖
- **OCR 拍照识字**：拍摄实体文字 → 识别 → 展示笔顺

#### 5.3.2 社交功能

- **书写分享**：录制书写过程 → 导出短视频/GIF → 分享至社交平台
- **班级群组**：教师创建班级 → 布置每日作业 → 查看完成情况
- **排行榜**：班级/家庭周榜，按正确率 + 完成量排名
- **亲子 PK**：家长 vs 孩子实时写字对战，各自打分

#### 5.3.3 平台适配

- **Widget 小组件**：iOS/Android 桌面「今日汉字」卡片
- **Apple Watch**：每日推送一字 + 释义，手腕上学习
- **Apple Pencil**：压感适配，模拟真实毛笔粗细变化

## 6. Onboarding 流程设计

```
启动 App
  │
  ├─ 欢迎页（品牌动画，2 秒）
  │
  ├─ Step 1：选择模式
  │    "你是谁在用？"
  │    [🧒 小朋友] [🎓 大朋友]
  │
  ├─ Step 2：选择字形
  │    "你想学哪种字？"
  │    [简体中文] [繁體中文] [日本語]
  │
  ├─ Step 3：选择级别（可跳过）
  │    简体 → 年级/HSK
  │    繁体 → 年级
  │    日语 → JLPT
  │
  ├─ Step 4：体验核心功能
  │    "说一个字试试！" → 语音输入 → 笔画动画
  │    "跟着写一遍！" → 跟写体验（不评分，纯鼓励）
  │
  ├─ Step 5：权限请求
  │    麦克风权限（语音输入需要）
  │    通知权限（每日提醒，可跳过）
  │
  └─ 进入首页
       儿童 → 今日任务大卡片
       成人 → 搜索框 + 每日推荐
```

> **设计原则**：3 步内进入核心体验，权限请求延后到功能需要时。

## 7. 音频与交互细节

### 7.1 音效设计

| 触发 | 音效 | 说明 |
|------|------|------|
| 落笔 | 轻柔毛笔触纸声 | 增强真实感 |
| 笔画完成 | 短促「叮」 | 正反馈 |
| 整字写完 | 撒花/星星音效 | 成就感 |
| 评分满分 | 欢呼声（儿童）/ 古筝短音（成人） | 差异化激励 |
| 写错提示 | 温柔「嗯？」（非刺耳警告） | 避免挫败感 |

### 7.2 鼓励语音（儿童模式）

- 写完一字："太棒了！你写得真好看！"
- 连续完成 5 字："哇，你今天好厉害呀！"
- 纠正笔顺："没关系，我们再试一次好不好？"
- 全部完成："今天的任务都完成啦，你是写字小达人！"

### 7.3 手势操作

| 手势 | 功能 |
|------|------|
| 左滑 | 下一个字 |
| 右滑 | 上一个字 |
| 双指捏合 | 缩放书写区域 |
| 长按汉字 | 弹出详情卡片（读音、释义、组词、部首） |
| 摇一摇 | 随机推荐一个字（发现模式） |
| 三指下滑 | 撤销最后一笔 |

## 8. 留存机制

### 8.1 打卡日历

- 日历视图展示连续打卡天数
- 连续打卡 7 天 → 解锁特殊徽章
- 连续打卡 30 天 → 解锁限定字体/主题

### 8.2 补签卡

- 每月赠送 1 张补签卡
- 付费用户额外赠送 2 张
- 断签后 48 小时内可使用

### 8.3 每周主题解锁

- 每周解锁一个主题字库（如：动物、食物、节日…）
- 配合主题插画和小故事
- 完成本周主题 → 解锁主题徽章

### 8.4 推送策略

| 时段 | 推送内容 | 频率 |
|------|----------|------|
| 早 8:00 | "早安，今天学点新字吧 ☀️" | 每日（可关闭） |
| 晚 19:00 | "今天还没打卡哦，就差 3 个字！" | 仅未完成时 |
| 周末 10:00 | "本周你学了 35 个字，比上周多 8 个 🎉" | 每周六 |

> **静默规则**：连续忽略 3 次推送 → 降频至每周 1 次；7 天无打开 → 发送挽回推送 1 次后静默。

### 8.5 成就墙

- **徽章体系**：集齐全部部首、完成 100/500/1000 字、连续打卡里程碑…
- **等级系统**：入门 → 初学 → 进阶 → 达人 → 大师
- **成就分享**：一键生成成就卡片 → 社交分享

## 10. 付费墙策略

### 10.1 核心转化路径

```
免费用户看到笔画动画 → "好酷，我也想写！"
  → 点击跟写 → 提示"解锁跟写打分，开始你的练字之旅"
  → 7 天免费试用 → 到期转化
```

### 10.2 付费墙触发点

| 触发场景 | 展示方式 |
|----------|----------|
| 点击「跟写」按钮 | 弹出试用/订阅引导 |
| 每日第 6 个字 | 柔性提示"今日免费额度已用完" |
| 查看进度报告 | 模糊展示 + 解锁引导 |
| 生成字帖 PDF | 免费仅预览首页，完整版需订阅 |

### 10.3 设计原则

- **免费版够用但不够爽**：能看不能写，天然痒点
- **不打断体验**：付费提示出现在自然断点
- **试用优先**：首次触发给 7 天免费试用，降低决策门槛
- **儿童保护**：儿童模式下付费操作需家长验证（密码/Face ID）

## 11. 视觉风格

### 11.1 儿童模式

- **风格**：手绘插画、圆角卡片、明亮配色
- **配色**：主色温暖橙 + 天蓝辅色 + 大量白色留白

- **动效**：弹性动画、粒子撒花、角色表情变化

- **风格**：水墨极简、留白克制
- **配色**：墨黑 + 宣纸白 + 朱红点缀
- **字体**：衬线体（如：思源宋体）
- **动效**：微妙渐变、水墨晕染
- **意境**：像翻开一本精致的字帖

### 11.3 笔画渲染

- 模拟毛笔质感：起笔尖、运笔粗、收笔回锋
- 支持 Apple Pencil 压感 → 笔画粗细实时变化
- 墨色渐变：起笔浓 → 中段稍淡 → 收笔浓
- 可选项：铅笔/钢笔/毛笔笔触切换

## 13. 数据与隐私合规

### 13.1 适用法规

| 法规 | 适用地区 | 关键要求 |
|------|----------|----------|
| **COPPA** | 美国 | 13 岁以下儿童数据保护，需家长可验证同意 |
| **GDPR-K** | 欧盟 | 16 岁以下（各国可降至 13 岁）需家长同意 |
| **个人信息保护法** | 中国 | 14 岁以下需监护人同意，单独告知 |
| **儿少法** | 台湾 | 儿童个资处理需法定代理人同意 |

### 13.2 数据架构原则

- **本地优先**：学习数据、书写记录默认存储在设备本地
- **最小收集**：仅收集必要数据（学习进度、匿名使用统计）
- **端到端加密**：云同步数据加密传输和存储
- **数据可删**：用户可随时导出/删除全部个人数据

### 13.3 家长同意机制

```
儿童模式首次注册 → 需填写家长邮箱
  → 发送验证邮件至家长
  → 家长点击确认链接 → 完成知情同意
  → 未验证状态下仅允许离线体验（无账号功能）
```

### 13.4 其他合规

- 儿童模式无第三方跟踪 SDK
- 广告合规：儿童模式仅展示经审核的品牌广告（无行为定向）
- App Store / Google Play 儿童类目审核要求全覆盖

## 16. 商业化路线图

### Phase 1：验证期（V1.0 上线后 0-3 个月）

- 目标：验证核心体验，获取首批 10,000 用户
- 收入：付费订阅（¥18/月、¥128/年）
- 渠道：ASO 优化、小红书/抖音内容营销、育儿社群投放
- 关键指标：次日留存 > 40%、7 日留存 > 20%、试用→付费转化 > 8%

### Phase 2：增长期（3-9 个月）

- 上线繁体中文 + 完整 P1 功能
- 推出家庭版（¥198/年）
- 增加社交分享功能，驱动口碑传播
- 目标：DAU 50,000、月收入 ¥200,000+
- 拓展台湾/日本/东南亚市场

### Phase 3：规模化（9-18 个月）

- 推出校园版 B 端
- 完成 P2 功能（字源动画、OCR、Widget…）
- IP 吉祥物设计 + 周边衍生（字帖本、文具）
- 目标：DAU 200,000、ARR ¥5,000,000+
- 寻求 Pre-A 轮融资

### Phase 4：生态化（18 个月+）

- 实体联动：定制字帖练习本、智能点读笔
- 平台化：开放字库 API，接入第三方教育 App
- 出海：英文/韩文界面，覆盖全球中文学习市场
- 目标：ARR ¥20,000,000+
