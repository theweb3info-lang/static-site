# NudgeMate — ADHD AI语音助理产品计划

## 📌 产品概述

**App名称建议：NudgeMate**
- "Nudge"（轻推）+ "Mate"（伙伴），友好、非临床感
- 备选：BuddyBuzz, RingMind, CallBuddy

**一句话定位：** AI会像朋友一样打电话提醒你，不是冷冰冰的闹钟。

**核心洞察：** ADHD大脑会无视推送通知（notification blindness），但接电话是一个需要主动回应的行为，打断力远强于被动通知。

## 🏗️ 核心功能

### MVP（3周）

1. **AI语音电话提醒**
   - 用户设定提醒 → 到时间AI打电话
   - AI用温暖友好的语气提醒（不是机器人音）
   - 用户必须接听并语音确认才算完成
   - 如果没接，间隔3分钟再打，最多3次

2. **智能提醒设置**
   - 简单的任务/提醒创建界面
   - 重复提醒（每日/每周）
   - 时间选择器

3. **基础个性化**
   - 选择AI语音风格（温柔朋友/活力教练/冷静助手）
   - 设置昵称，AI会叫你的名字

4. **用户账户系统**
   - 手机号注册（需要手机号来打电话）
   - 基础设置页面

### V2（第2-3个月）

1. **AI对话式提醒**
   - AI不只是提醒，还能帮你分解任务
   - "你说要打扫房间，要不我们先从收拾桌子开始？"
   
2. **情绪感知**
   - AI通过语音语调判断你的状态
   - 如果听到你很烦躁，会调整沟通方式

3. **accountability buddy模式**
   - AI在30分钟后回拨确认你是否完成了
   - 温和的追踪，不是压力

4. **每周复盘报告**
   - 完成率统计
   - 哪些时间段你最容易错过提醒
   - AI建议优化提醒时间

5. **紧急联系人升级**
   - 如果AI打3次你都没接，可以通知你指定的accountability partner

6. **日历集成**
   - 同步Google Calendar / Apple Calendar
   - 自动为日历事件创建电话提醒

## 🔧 技术架构

### 整体架构

```
Flutter App (iOS/Android)
    ↕ REST API
Backend (Node.js / Python FastAPI)
    ├── Twilio Voice API (打电话)
    ├── OpenAI / ElevenLabs (AI语音生成)
    ├── Firebase (认证、推送)
    ├── PostgreSQL (数据存储)
    └── Redis (任务队列、调度)
```

### 语音电话实现详情

**Twilio Voice API：**
- 使用Twilio Programmable Voice发起出站电话
- TwiML控制通话流程
- 流程：调度器触发 → 后端调Twilio API → Twilio打电话给用户 → 播放AI语音 → 等待用户语音回应 → 结束

**AI语音方案：**
- **方案A（推荐）：ElevenLabs + Twilio**
  - ElevenLabs生成自然语音音频
  - 预生成常用提醒语音模板，动态部分实时生成
  - 通过Twilio `<Play>` 播放音频
  
- **方案B：Twilio + OpenAI Realtime API**
  - 实时对话式，更自然但成本更高
  - 适合V2的对话式提醒

**成本估算（每次通话）：**

| 项目 | 成本 |
|------|------|
| Twilio出站电话（美国） | $0.014/分钟 |
| Twilio电话号码 | $1.15/月/号码 |
| ElevenLabs语音生成 | ~$0.003/次（预生成模板） |
| OpenAI（文本生成提醒内容） | ~$0.001/次 |
| **每次提醒总成本** | **~$0.02-0.03**（30秒通话） |

**用户经济模型：**
- 假设用户每天3次提醒 = 每月90次
- 每月成本 = 90 × $0.025 = $2.25
- 月费 $4.99 → **毛利率 ~55%**
- 年费 $39.99/12 = $3.33/月 → **毛利率 ~32%**（可接受，因为年费留存率高）

### 调度系统

```
用户创建提醒 → 存入DB → Redis Sorted Set (按时间排序)
                              ↓
                    Worker每秒检查到期任务
                              ↓
                    触发Twilio API发起通话
                              ↓
                    通话状态回调 → 更新DB
                              ↓
                    未接听 → 3分钟后重试（最多3次）
```

### Flutter App架构
- **状态管理：** Riverpod
- **本地存储：** Hive / SharedPreferences
- **网络：** Dio + Retrofit
- **推送：** Firebase Cloud Messaging（作为电话前的预提醒）

## 🤖 AI个性化策略

### 提醒内容个性化
- AI不会每次说一样的话，基于：
  - 任务类型（吃药 vs 开会 vs 锻炼）
  - 时间段（早上活力 vs 晚上温柔）
  - 用户历史响应模式
  - 当天已完成/未完成的任务数

### 语音风格选择
1. **☀️ 阳光朋友** — 活泼热情，适合需要能量的人
2. **🌙 温柔教练** — 平静温暖，适合容易焦虑的人  
3. **⚡ 直接了当** — 简短有力，适合不喜欢废话的人

### 示例提醒对话
```
AI: "嘿 Alex！是我，NudgeMate。你25分钟后有个跟牙医的预约，
     现在是个好时间开始准备出门了。需要我30分钟后再打给你
     确认你出发了吗？"
     
用户: "好的谢谢"

AI: "没问题！祝你看牙顺利，回头聊！"
```

## ⚖️ 监管考量

### 非医疗设备声明
- **NudgeMate不是医疗设备，不提供医疗建议**
- 定位为"生产力/提醒工具"，不是ADHD治疗工具
- 避免使用"治疗"、"改善症状"等医疗声明
- 推荐语：*"Designed for busy brains"*而不是*"Treats ADHD"*

### 合规要点
- **HIPAA：** 不收集健康数据，不涉及HIPAA
- **TCPA（电话消费者保护法）：** 用户主动注册+明确同意接听 = 合规
- **COPPA：** 13岁以上用户，无需特殊处理
- **App Store审核：** 不归类为健康App，避开严格审核
- **隐私政策：** 需明确说明电话号码用途、通话录音策略（建议不录音）

### Terms of Service关键条款
- 用户同意在设定时间接收电话
- 用户可随时取消
- 免责声明：不对错过重要事项负责
