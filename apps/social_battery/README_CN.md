# 🔋 社交电量管理器 (Social Battery)

> 内向者的社交能量管理工具

## 📱 产品简介

社交电量管理器是一款专为内向者设计的 App，帮助你可视化管理社交能量。通过电池图标直观展示你的"社交电量"，记录社交活动，智能建议休息时间，还提供话题卡片和退场借口。

### 用户画像
- 内向者 / 社恐人群
- 需要独处时间充电的人
- 想要量化管理社交精力的人

## ✨ 核心功能

### 1. 社交电量仪表盘 🔋
- 可视化电池图标（CustomPainter 绘制）
- 电量从 100%→0%，带动画效果
- 根据电量给出状态建议
- 快速充电/消耗按钮

### 2. 手动记录社交活动 📝
- 10种社交活动类型
- 自定义时长和能量消耗
- 添加备注记录感受
- 预览消耗效果

### 3. 社交日历 📅
- 本周社交负荷柱状图（fl_chart）
- 活动列表管理
- 空状态温暖提示

### 4. 话题卡片 💬
- 30+ 精选 small talk 话题
- 卡片翻转动画
- 一键复制分享

### 5. 退场借口生成器 🏃
- 25+ 有趣实用的退场借口
- 随机抽取功能
- 一键复制

### 6. 每周社交报告 📊
- 社交次数/总消耗/时长统计
- 活动类型排行
- 个性化洞察建议

## 🛠 技术栈

- **框架**: Flutter 3.38
- **状态管理**: Riverpod
- **本地存储**: sqflite
- **图表**: fl_chart
- **电池动画**: CustomPainter

## 📁 项目结构

```
lib/
  app/          — App入口、主页
  features/     — 功能模块
    dashboard/  — 电量仪表盘
    activities/ — 社交活动记录
    calendar/   — 社交日历
    topics/     — 话题卡片
    excuses/    — 退场借口
    report/     — 周报
  shared/       — 共享模块
    theme/      — 主题配置
    utils/      — 工具/Provider
    constants/  — 常量数据
```

## 🚀 构建运行

```bash
# 获取依赖

# 运行调试
flutter run

# 构建 APK
flutter build apk --release
```
