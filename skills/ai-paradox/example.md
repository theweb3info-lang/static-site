2. **Research data** (20 min)
   - Find 5-7 authoritative sources
   - Note exact percentages and quotes

3. **Structure 3 layers** (30 min)
   - Brainstorm 3 angles
   - Each layer needs 2-3 sub-points

4. **Write solutions** (15 min)
   - Find real frameworks (McKinsey, MIT, Bain)
   - Make them actionable

5. **Polish + add metaphors** (15 min)
   - Replace abstract with concrete
   - Add 2-3 analogies

6. **Data summary** (10 min)
   - Pull out key stats
   - Format cleanly

**Total: ~100 minutes for a 3000-word article**
